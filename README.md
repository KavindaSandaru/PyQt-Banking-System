## 📌 Overview

PyQt Banking System is a desktop banking application designed to simulate real-world banking operations. The application provides a modern user experience with secure account management, balance tracking, transaction history, money transfers, and profile customization.

This project was developed to improve desktop application development skills using Python and PyQt6 while implementing banking system concepts and user interface design principles.

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/KavindaSandaru/PyQt-Banking-System.git
cd PyQt-Banking-System
```

### 2. Create Virtual Environment

```bash
python -m venv venv
```

### 3. Activate Virtual Environment

Windows:

```bash
venv\Scripts\activate
```

Linux / macOS:

```bash
source venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install PyQt6
```

### 5. Run the Application

```bash
python main.py
```
